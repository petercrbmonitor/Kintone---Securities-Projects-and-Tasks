# Task Assignment - Quick Reference

## Creating Tasks

### From a Single Record (App 23)
1. Open record → Click **🚩 Create Task**
2. Pick template or type name
3. Select assignee → Set due date
4. Click **Create Task & Notify**

### From a Filtered View (App 23)
1. Filter records needing work
2. Click **🚩 Create Task from View**
3. Name the batch task
4. Enter record count → Select assignee
5. Click **Create Task & Notify**
